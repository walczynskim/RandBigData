Praca domowa odnośnie pakietu archivist

W pierwszej kolejności wczytywane są dane

library(archivist) 
library(ggplot2)
library(knitr) #wszystkie 3 biblioteki potrzebne
library(Cairo) #Windows i pdfy nie zawsze się lubią
options(OutDec=",") #przecinek jako separator dzisiętny
iris <- iris #Nieśmiertelny Fisher
attach(iris)

Repozytorium

W kolejnym kroku trzeba stworzyć repozytorium, w którym trzeba zgromadzić dane

repo <- "wykresy"
createLocalRepo(repoDir = repo, default = TRUE)

Eksport hashów

Podaję ścieżkę eksportu hashów na github.

addHooksToPrint(class=c("ggplot", 
                        "data.frame"),
                repoDir = "wykresy", 
                repo = "Eseje", 
                user = "pbiecek", 
                subdir = "arepo") #nie wiem, czy działa

Tworzenie wykresów

Korzystając z pakietu ggplot2 oraz danych iris tworzę cztery wykresy.

Pierwszy wykres

\(\ \\ \)

ggplot(iris, 
       aes(Petal.Width, 
           Petal.Length, 
           group=Species
           )
       )+
  geom_point(aes(color=Species, 
                 shape=Species
                 ),
             size=I(2)
             )+
  labs(title ="Płatki", #petal=płatek
       x = "Szerokość płatka", 
       y = "Długość płatka"
       )+
  scale_fill_manual(name="Gatunek" #Dlaczego nie działa?
                    )
